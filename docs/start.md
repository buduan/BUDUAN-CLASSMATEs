## 快速上手
### 下载程序
目前未开放下载！
### 部署在服务器中
将下载好的程序部署在服务器中，部署成功后，访问 **您的域名/install** 根据安装程序的引导进行安装即可。
### 手动安装
打开您保存程序的目录中的**includes**目录，将其中的数据库信息修改成自己服务器中的信息，并导入程序根目录中的db.sql即可。
